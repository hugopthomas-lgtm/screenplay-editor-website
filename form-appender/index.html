Form Appender

Automatically append Google Forms responses to Google Docs. Set it up once, and every submission lands in the right document.

Install from Marketplace

How it works

1

Pick a Drive folder

Select a folder containing the Google Docs where responses should be appended.

2

Map a doc-picker question

Choose which form question determines the target Doc. The choices are auto-populated from your folder.

3

Select questions & format

Pick which responses to include and choose your output format: bullets, vertical table, or horizontal table.

4

Activate the trigger

Install the trigger and you're done. Every new submission is automatically appended to the right Doc.

Three output formats

*

Bullets

Each submission as a clean bulleted list with timestamp and respondent info.

||

Vertical table

One table per submission, with question-answer rows. Great for detailed entries.

=

Horizontal table

One growing table with a row per submission. Perfect for tracking over time.

Free & private

Form Appender is free to use. All processing happens within your Google account. No external servers, no data collection, no tracking.
