Mode d'emploi KDP

Mes Tout Premiers Classiques — Pouvoir Magique Editions

1. Fichiers a preparer

Interieur du livre (PDF)

Export Canva avec l'option "Crop marks and bleed" cochee. Aucun probleme constate cote KDP.

Couverture (PDF)

Erreur a eviter : Ne PAS exporter avec "Crop marks and bleed" depuis Canva. KDP refuse les fichiers avec les reperes de coupe (les petites fleches dans les coins).
Bonne methode : Creer le design Canva directement aux dimensions totales fournies par KDP (qui incluent deja les 3,175 mm de fond perdu de chaque cote). Puis exporter en PDF sans cocher l'option bleed/crop. Le fichier final doit etre "plein pot" aux bonnes dimensions, sans marques.

En resume : KDP veut le fond perdu integre dans le fichier, mais pas les marques visuelles de coupe.

2. Creation du livre dans Canva

Format

Creer l'album au format 21,59 x 21,59 cm (format carre standard KDP)

Fonds et mise en page

  • Fond toujours plus etendu que la page — chaque illustration de fond doit deborder pour eviter les liseres blancs a l'impression
  • Varier les mises en page — les images qui ne prennent pas toute la page sont appreciees
  • Environ 3 doubles pages par livre — pour creer des moments forts visuellement
  • Finir sur une double page — c'est la marque de fabrique de la collection

Texte

  • Hauteur constante — ecrire le texte toujours a la meme hauteur sur chaque page, sauf exception justifiee (composition particuliere)
  • Pas de guillemets — meme pour les dialogues

3. Generation des images (Midjourney)

Visages et expressions

Utiliser Nanobanana pour les visages, afin que les expressions varient d'une illustration a l'autre (eviter le syndrome "meme tete partout").

Couverture

Titre genere par ChatGPT Image — utiliser la generation d'image de ChatGPT pour creer le titre de couverture (meilleur rendu typographique).

5. Creation de la version Kindle (ebook)

Fichiers a exporter depuis Canva

1. Couverture

  • Exporter uniquement le recto (pas le dos ni la tranche)
  • Format : JPG
  • Sans fond perdu (bleed)

2. Manuscrit

  • Exporter en PDF Print
  • Sans fond perdu (bleed)
  • Pages interieures uniquement (sans la couverture)

Conversion en KPF

  • Ouvrir Kindle Create
  • Choisir "Livre pour enfants"
  • Importer le PDF du manuscrit
  • Exporter en .kpf

Publication sur KDP

  • Aller sur la version brochee existante
  • Cliquer sur "Creer une version Kindle"
  • Charger le fichier .kpf
  • Charger la couverture JPG
  • Inscrire a KDP Select (pour promos 5 jours gratuits)
  • Prix : 8,99 € — 35% de redevance (plus avantageux pour fichiers lourds)
Ne PAS cocher "Domaine public" — meme si l'histoire originale est dans le domaine public, notre version (illustrations, adaptation du texte) est une creation originale protegee.
