Privacy Policy

Last updated: December 2025

Screenplay Editor does not collect, store, or transmit any personal data to external servers. All data stays within your Google account.

Information We Access

To provide its functionality, Screenplay Editor requires access to:

1. Current Google Document

We access the content of your open document to apply screenplay formatting and analyze script structure.

Scope: Limited to the document you have open. We cannot access other documents.

2. Google Drive (Export)

When you export, we create PDF or Fountain files in your Drive.

Scope: We only create new files. We don't read, modify, or delete other files.

3. User Interface

We display a sidebar within Google Docs. This is a standard add-on capability.

Scope: Standard Google Workspace add-on behavior.

Information We Store

User Preferences

Settings like formatting options are stored via Google's PropertiesService, within Google's infrastructure, associated with your account, not accessible to us.

Document-Specific Data

Info like your character list is stored in the document's properties. This data travels with the document and is only accessible to users with document access.

What We Don't Collect

Document content to external servers
Your scripts outside Google
Personal info (name, email)
Analytics or tracking
Third-party data sharing
Advertising data

Data Security & Retention

All data remains within Google's infrastructure, protected by Google's security. We don't operate external servers, so we can't be a point of data breach.

To delete all data, simply uninstall the add-on from your Google account.

Google API Compliance

Screenplay Editor's use of information received from Google APIs adheres to the Google API Services User Data Policy, including Limited Use requirements.
